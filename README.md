# InvoiceBook

Desktop εφαρμογή (Electron) για καταχώρηση και παρακολούθηση τιμολογίων προμηθευτών.

## Τι κάνει

- Καταχώρηση προμηθευτών και τιμολογίων (με γραμμές ειδών, ΦΠΑ, ποσά).
- Επισύναψη σαρωμένου PDF σε κάθε τιμολόγιο.
- Μαζική εισαγωγή από αρχείο CSV ή JSON, με στάδιο επιβεβαίωσης πριν οι εγγραφές γίνουν οριστικές.
- Συνοπτικές αναφορές ανά μήνα/έτος.

Η εφαρμογή και το περιβάλλον χρήστη είναι στα ελληνικά.

## Απαιτήσεις

- [Node.js](https://nodejs.org/) (για το Electron)
- Python 3 στο PATH (`python` σε Windows, `python3` αλλού) — χρησιμοποιείται μόνο η standard library, καμία εξωτερική εξάρτηση.

## Εκτέλεση

```bash
npm install
npm start
```

Δεν υπάρχει ξεχωριστό βήμα build· το Electron φορτώνει τα αρχεία απευθείας.

## Αρχιτεκτονική

Σύντομα: η εφαρμογή τρέχει σε δύο διεργασίες — το Electron main process (`main.js`) ξεκινάει ένα μόνιμο Python subprocess (`backend/bridge.py`) και επικοινωνούν μέσω JSON γραμμών πάνω από stdio. Η επιχειρησιακή λογική και η πρόσβαση στη SQLite βάση βρίσκονται στο `backend/database.py`. Λεπτομέρειες σχεδιασμού, συμβάσεις, και σημεία προσοχής τεκμηριώνονται στο [`CLAUDE.md`](./CLAUDE.md).

## Κατάσταση

Έκδοση v0.1 (development). Σε packaged builds ο φάκελος δεδομένων (βάση SQLite + αποθηκευμένα PDF) βρίσκεται στον φάκελο χρήστη (`app.getPath('userData')`)· κατά την ανάπτυξη (`npm start`) παραμένει μέσα στο `backend/` για εύκολη επιθεώρηση/επαναφορά. Το packaging για διανομή γίνεται με `npm run dist` (electron-builder).
